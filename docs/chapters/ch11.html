技能系統 (Skills)

可選注入的專業知識 — skills.rs

技能 vs 命令

維度命令 (Commands)技能 (Skills)
格式TOMLMarkdown (SKILL.md)
觸發方式主動呼叫 (/command)選擇注入
注入位置替換使用者輸入追加到 System Prompt
用途定義任務模板提供專業知識上下文

技能目錄結構

技能可以是單一 .md 檔案，或包含 SKILL.md 的目錄：

技能目錄結構 
.gcd/skills/
├── rust-patterns.md           ← 單檔技能
├── security-audit/
│   └── SKILL.md               ← 目錄技能
└── api-design/
    └── SKILL.md               ← 目錄技能

技能的第一個非空行會被擷取為摘要描述，供選擇時顯示。完整內容在選中後注入到 prompt 中。

原始碼：crates/gcd-core/src/skills.rs
