容器化沙箱

Docker/Podman 隔離執行 — tools/container.rs

Container 模式運作

當 Sandbox 設為 container 時，shell 工具的命令會在隔離容器中執行：

  • 工作區目錄掛載到容器內
  • 容器內無法存取主機其他檔案系統
  • 網路存取可配置（預設隔離）
  • 支援 Docker 和 Podman 兩種 runtime
適用場景

Container 模式適用於：審查不信任的第三方 PR、執行可能有操作影響的程式碼、在 CI/CD 中安全執行 Agent。

原始碼：crates/gcd-core/src/tools/container.rs
