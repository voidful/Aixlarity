工具系統 (Tool System)

Agent 與現實世界互動的橋梁 — tools.rs

Tool Trait

tools.rs — Tool trait 定義 
#[async_trait]
pub trait Tool: Send + Sync {
    fn name(&self) -> &str;
    fn description(&self) -> &str;
    fn parameters_schema(&self) -> serde_json::Value;
    async fn execute(
        &self,
        params: serde_json::Value,
        ctx: &ToolContext,
    ) -> Result<String>;
}

8 個 built-in tools

read_file
讀取指定檔案的內容。支援相對路徑（基於工作區根）和絕對路徑。
沙箱：受讀取範圍限制
write_file
建立或覆寫檔案。自動生成 unified diff 預覽，供使用者審核。
沙箱：受寫入範圍限制 | 權限：suggest 模式需確認
list_dir
列出目錄內容，截斷至 200 筆以避免過大輸出。
沙箱：受讀取範圍限制
shell
執行 shell 命令。輸出截斷至 10KB。在 container 模式下命令在 Docker 容器內執行。
沙箱：受隔離策略限制 | 權限：suggest/auto-edit 需確認
search_files
使用 ripgrep (rg) 搜尋檔案內容。支援正則表達式和檔案類型過濾。
沙箱：受讀取範圍限制
fetch_url
HTTP GET 抓取網頁內容。內建簡易 HTML → Markdown 轉換。
沙箱：不受檔案系統限制
apply_patch
套用 unified diff 格式的修補。比 write_file 更精準，適合小範圍修改。
沙箱：受寫入範圍限制 | 權限：suggest/auto-edit 需確認
