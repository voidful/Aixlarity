核心依賴

依賴版本用途選擇理由
tokio1.x非同步 RuntimeRust 生態事實標準，full features
reqwest0.11HTTP 客戶端支援 streaming、TLS、連接池
serde1.0序列化框架Rust 生態標準，derive 巨集
serde_json1.0JSON 處理搭配 serde 使用
clap4.xCLI 解析Derive 巨集、自動生成 help
rustylinelatestREPL 行編輯歷史、補全、Ctrl+R
colored2.x終端色彩輕量、API 簡潔
thiserror1.0錯誤定義derive Error trait
anyhow1.0錯誤傳播快速原型、context 鏈
async-trait0.1非同步 TraitTool trait 的非同步支援
futures-util0.3Stream 組合子SSE 串流處理

兩個 Crate 的職責

L

gcd-core (Library)

所有業務邏輯。可被其他 Rust 程式作為函式庫引用。零 CLI 依賴、零 UI 邏輯。純粹的 Agent 引擎。

B

gcd-cli (Binary)

薄殼 CLI。只負責解析命令列參數、啟動 REPL、呼叫 gcd-core。約 500 行。
