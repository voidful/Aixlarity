指令載入器

AGENTS.md 如何變成 System Prompt — instructions.rs

指令檔案搜尋順序

Aixlarity 在工作區根目錄搜尋以下指令檔，全部載入後合併：

  1. AGENTS.md — 主指令檔（所有供應商共用）
  2. AIXLARITY.md — Aixlarity 專屬指令
  3. GEMINI.md — 僅在使用 Gemini 供應商時載入
  4. CLAUDE.md — 僅在使用 Anthropic 供應商時載入
信任閘門

指令載入受 Trust System 控制。如果當前工作區被標記為 untrusted，所有專案指令檔都不會被載入。這是防禦 Prompt Injection 的核心機制。

注入到 System Prompt 的位置

載入的指令會被包裝成特定格式，插入到最終提示詞的 system message 中：

最終 System Prompt 結構 
[內建 Agent 角色描述]
[工具能力說明]
---
[AGENTS.md 內容]
[AIXLARITY.md 內容]
[供應商特定指令]
---
[注入的技能內容]
