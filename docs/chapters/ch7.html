多供應商系統

如何將三大 LLM API 統一為一個介面 — providers.rs

Provider 資料模型

providers/types.rs 
pub struct ProviderProfile {
    pub id: String,              // "gemini-env"
    pub label: String,           // "Gemini (環境變數)"
    pub family: ProviderFamily,  // Gemini | OpenAi | Anthropic
    pub protocol: ProviderProtocol, // API 通訊協議
    pub model: String,           // "gemini-2.5-pro"
    pub base_url: String,        // API 端點
    pub api_key_env: String,     // 金鑰環境變數名
}

pub enum ProviderProtocol {
    GeminiStreamingRest,    // Gemini 原生 SSE
    OpenAiResponses,        // OpenAI Responses API
    AnthropicMessages,      // Anthropic Messages API
    OpenAiChat,             // 通用 OpenAI 相容
}

pub enum ProviderFamily {
    Gemini,
    OpenAi,
    Anthropic,
    OpenAiCompatible,       // OpenRouter, Ollama 等
}
