信任模型 (Trust System)

誰可以控制 Agent 的行為 — trust.rs

三級信任模型

Trusted

完全信任

載入 .aixlarity/ 配置、AGENTS.md 指令、自訂命令與技能。適用於你自己的專案。

Parent-Trusted

繼承信任

從父目錄繼承信任設定。適用於 monorepo 中的子專案。

Untrusted

不信任

拒絕載入任何專案配置。僅使用全域設定和內建工具。適用於檢視第三方 PR。

信任規則儲存

~/.aixlarity/trusted-folders.txt 
trusted	/Users/dev/my-project
parent	/Users/dev/monorepo
untrusted	/tmp/random-pr-clone

安全意義

信任系統防止惡意 AGENTS.md 注入。如果你 clone 了一個不信任的 repo，其中的指令檔案不會被自動載入。這避免了 Prompt Injection 攻擊。
