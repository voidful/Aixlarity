自訂供應商

在 ~/.gcd/providers.conf 或專案的 .gcd/providers.conf 中定義自訂供應商：

~/.gcd/providers.conf 
# 使用 OpenRouter 等相容 API
[my-openrouter]
label = "OpenRouter Claude"
family = "openai-compatible"
protocol = "openai-chat"
model = "anthropic/claude-sonnet-4-20250514"
base_url = "https://openrouter.ai/api/v1"
api_key_env = "OPENROUTER_API_KEY"

# 本機部署的模型
[local-ollama]
label = "Local Ollama"
family = "openai-compatible"
protocol = "openai-chat"
model = "codellama:34b"
base_url = "http://localhost:11434/v1"

供應商診斷

terminal 
# 檢查所有供應商狀態
gcd providers doctor

# 輸出範例：
# ✅ gemini-env: API key found (GEMINI_API_KEY)
# ✅ openai-env: API key found (OPENAI_API_KEY)
# ❌ claude-env: Missing ANTHROPIC_API_KEY

# 切換當前使用的供應商
gcd providers use gemini-env
