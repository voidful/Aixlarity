命令目錄系統

自訂斜線命令 — commands.rs

命令定義格式

命令以 TOML 格式定義，放置於 .gcd/commands/ 或 ~/.gcd/commands/：

.gcd/commands/test.toml 
description = "執行測試並修復失敗"
prompt = """
請執行 cargo test，分析所有失敗的測試，
然後逐一修復。修復後重新執行測試確認。
"""

命令呼叫語法

使用 /命令名 [參數] 語法呼叫。參數會附加到命令的 prompt 之後：

terminal 
# 呼叫 review 命令，附帶目標檔案
gcd exec "/review src/agent.rs"

# 呼叫 test 命令
gcd exec "/test"

載入優先順序

工作區命令 (.gcd/commands/) 優先於全域命令 (~/.gcd/commands/)。同名時工作區版本覆蓋全域版本。

原始碼：crates/gcd-core/src/commands.rs、crates/gcd-core/src/prompt/command_invocation.rs
