第 08 章

供應商設定檔格式

自訂供應商的 .conf 檔結構 — providers/config_loader.rs

.conf 檔案格式

供應商設定檔使用 INI 風格的格式（以 [section] 分隔），每個 section 是一個供應商設定。支援兩個位置：全域 ~/.gcd/providers.conf 和工作區 .gcd/providers.conf。

完整欄位參考

欄位必填說明範例值
label否顯示名稱"My OpenRouter"
family是供應商家族gemini / openai / anthropic / openai-compatible
protocol是API 協議gemini-streaming / openai-responses / anthropic-messages / openai-chat
model是模型名稱"gemini-2.5-pro"
base_url是API 端點"https://api.openai.com/v1"
api_key_env否金鑰環境變數"OPENROUTER_API_KEY"
原始碼：crates/gcd-core/src/providers/config_loader.rs
