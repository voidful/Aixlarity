組態管理與路徑系統

路徑解析、偏好設定、環境變數 — config.rs

AppPaths 結構

config.rs 定義了 Aixlarity 使用的所有重要路徑：

路徑說明位置
config_home全域配置根~/.aixlarity/
sessions_dir會話儲存~/.aixlarity/sessions/
commands_dir全域命令~/.aixlarity/commands/
skills_dir全域技能~/.aixlarity/skills/
plugins_dir全域插件~/.aixlarity/plugins/
providers_conf供應商設定~/.aixlarity/providers.conf
trusted_folders信任清單~/.aixlarity/trusted-folders.txt

SandboxPolicy 枚舉

config.rs — SandboxPolicy 
pub enum SandboxPolicy {
    Off,             // 不做限制
    ReadOnly,        // 只讀工作區
    WorkspaceWrite,  // 只寫工作區內 (預設)
    Container,       // Docker/Podman 隔離
}

環境變數覆蓋

環境變數覆蓋項目合法值
AIXLARITY_PROVIDER預設供應商 ID任何已知供應商 ID
AIXLARITY_SANDBOX沙箱策略off / read-only / workspace-write / container
AIXLARITY_TRUST是否啟用 trust 機制true / false / on / off / 1 / 0
容易誤會的一點

AIXLARITY_TRUST 不是直接指定 trusted 或 untrusted，而是控制是否啟用 trust 評估。真正的信任狀態來自 trusted-folders.txt 內的規則與目前路徑比對結果。
