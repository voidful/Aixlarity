Clap 4 命令結構

GCD 使用 Clap 4 的 derive 巨集定義命令結構。所有子命令映射到 app.rs 的 Facade 方法：

命令層級 
gcd
├── (default)          → 啟動 REPL
├── exec <prompt>      → 執行 Agent
├── overview           → 顯示工作區概覽
├── providers
│   ├── list           → 列出供應商
│   ├── current        → 顯示當前供應商
│   ├── show <id>      → 顯示供應商詳情
│   ├── use <id>       → 設定供應商
│   └── doctor         → 診斷供應商狀態
├── trust
│   ├── status         → 顯示信任狀態
│   └── set <level>    → 設定信任等級
├── sessions
│   ├── list           → 列出會話
│   ├── show <id>      → 顯示會話詳情
│   ├── fork <id>      → 分支會話
│   └── replay <id>    → 回放會話
├── checkpoints
│   └── list           → 列出檢查點
└── commands
    └── reload         → 重載命令/技能

REPL 實作

互動模式使用 rustyline 提供行編輯功能：

  • 上下方向鍵瀏覽歷史
  • Ctrl+R 反向搜尋歷史
  • 歷史自動持久化到 ~/.gcd/history.txt
  • 空行或 Ctrl+D 退出
