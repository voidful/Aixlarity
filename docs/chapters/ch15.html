Fork / Resume / Replay

R

Resume（恢復）

繼續同一個 session，在現有對話歷史後追加新輪次。aixlarity exec --resume <id>

F

Fork（分支）

複製現有 session 到新 ID，從此獨立發展。aixlarity exec --fork <id>
