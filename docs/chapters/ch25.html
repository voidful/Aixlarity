Undercover 模式

Aixlarity 支援 Undercover 模式，讓它能在 Gemini CLI 或 Claude Code 的工作區配置下運作。偵測到 .gemini/ 或 .claude/ 配置目錄時，Aixlarity 會自動適配對應的指令格式。
