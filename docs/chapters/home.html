核心工作台

MC

Mission Control

多 workspace、多 task、多 agent 的任務控制台，支援 pause、resume、cancel、retry 與背景長任務追蹤。

AR

Artifact Review

Implementation Plan、Task List、Diff、Test Report、Screenshot、Browser Recording 都是可留言、可批准、可重生的 artifact。

BR

Integrated Browser Agent

Managed browser 不只截圖，也能收 DOM、console、network、screenshot、video，並把 browser state 餵回 prompt。

TR

Terminal Replay

命令 ownership、stdout、stderr、exit code、cwd、env 摘要都可附到 artifact，避免「測過但沒有證據」。

DF

Visual Diff Review

詳細視覺化每次 AI 修改，支援逐檔、逐 hunk、逐輪確認，讓 code review 成為 agent workflow 的核心。

PV

Provider Control Center

OpenAI、Anthropic、Gemini 與相容端點可依 user/workspace scope 切換；provider 必須自己填模型，API key 不進匯出檔。

KL

Knowledge Ledger

把 continuous learning 做成可審核規則表：rules、memory、workflow、MCP activation 都能匯出與關閉。

EA

Editor-native Actions

diagnostic hover、Problems panel、selection、terminal output 都能直接送給 agent，不需要離開當前工作上下文。
