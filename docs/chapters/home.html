核心能力

MC

Mission Control

多個工作區、多個任務、多個 AI agent，都有狀態、控制與恢復點。

BR

Browser Evidence

DOM、console、network、截圖、影片都能成為瀏覽器證據。

TR

Terminal Replay

每個命令都保留 stdout、stderr、exit code、執行路徑與負責任務。

DF

Visual Diff Review

逐檔、逐段、逐輪確認 AI 修改，審完再合併。

PV

Provider Control

OpenAI、Anthropic、Gemini 與相容端點可依工作區切換。

KL

Knowledge Ledger

學到的規則、記憶、工作流程都可審核、匯出、關閉。

從原始碼啟動 IDE

想從 source build 也可以。Release 由 CI 出包；開發者路徑則保留 Rust runtime 與 VS Code fork 的完整啟動方式。

cargo build --release -p aixlarity
cd aixlarity-ide
npm run electron
npm run test-aixlarity-submission

一句話定位

Aixlarity IDE 是開源 AI agent 工作台。它把模型、工具、權限、證據、diff、模型供應商和教學都放在同一個可觀察的產品介面裡。
