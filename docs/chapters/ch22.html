Coordinator 機制

靈感來自 Claude Code 的 Agent tool 與 Claurst 的 Coordinator。Aixlarity 的協調器允許主 Agent 生成子 Agent 來處理獨立子任務：

  • 主 Agent 識別可並行處理的子任務
  • 每個子 Agent 擁有獨立的 context window
  • 子 Agent 繼承主 Agent 的 Sandbox 和 Permission 設定
  • 結果匯總回主 Agent 繼續處理
原始碼：crates/aixlarity-core/src/tools/coordinator.rs
