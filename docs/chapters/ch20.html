MCP (Model Context Protocol) 概述

MCP 是 Anthropic 提出的開放標準，讓 AI Agent 能動態發現和呼叫外部工具。Aixlarity 實作了 MCP 客戶端，透過 stdio 傳輸層與 MCP 伺服器通訊。

運作流程

1
讀取 .aixlarity/mcp.json 配置
2
啟動 MCP 伺服器子程序（例如 npx @modelcontextprotocol/server-filesystem）
3
透過 JSON-RPC 2.0 發送 tools/list 請求，發現可用工具
4
將 MCP 工具包裝為 Aixlarity 的 Tool trait 物件
5
Agent 呼叫 MCP 工具時，透過 JSON-RPC 轉發到伺服器
原始碼：crates/aixlarity-core/src/mcp.rs
